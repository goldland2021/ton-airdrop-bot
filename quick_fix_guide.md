### 监控脚本：
```batch
:: monitor_bot.bat
@echo off
:loop
tasklist | findstr "python.exe" > nul
if errorlevel 1 (
    echo Bot已停止，重新启动...
    cd /d "C:\Users\Administrator\.openclaw\workspace\TON_Airdrop_Bot"
    start python run_bot.py
)
timeout /t 60 /nobreak > nul
goto loop
```

## ⚠️ 风险提示

### 本地运行风险：
```
1. 突然断电导致服务中断
2. Windows更新自动重启
3. 网络波动影响连接

5. 安全风险（暴露到公网）
```
